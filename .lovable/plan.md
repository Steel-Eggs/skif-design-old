## 2. Hero-секция и шапка (десктоп + мобайл)

Основная цель — каталог должен бросаться в глаза.

### Десктоп

- Hero становится двухколоночным:
  - **Левая колонка** (~30–35% ширины): крупный блок «Каталог техники» — заголовок + вертикальный список категорий (Прицепы легковые, Прицепы для лодок, Прицепы для снегоходов, Грузовые прицепы, Запчасти и аксессуары и т.д.) со стрелочками и иконками. Каждая категория — крупный кликабельный пункт ~56–64px высотой.
  - **Правая колонка** (~65–70%): нынешний слайдер heroes (заголовок + описание + кнопки + статистика + изображение слайда). Компактнее, но визуально остаётся главным акцентом.
- **Шапка**:
  - Убрать выпадающее мега-меню «Каталог» (`CatalogMegaMenu`) из десктоп-навигации.
  - На освободившееся место в шапке поставить **строку поиска по сайту** (input + кнопка с лупой), которая ведёт на `/search?q=...` (страница `SearchResults` уже есть).

### Мобайл

- В hero сверху — крупный блок «Каталог» (одна большая кнопка-заголовок + сетка категорий 2 колонки с иконками, тапабельные плитки крупного размера).
- В бургер-меню оставить только основные ссылки (Главная, О компании, Услуги, Новости, Контакты, Корзина, Избранное) — **без подменю каталога**.
- Поиск в мобильной шапке — иконка-лупа, разворачивающаяся в полноширинный input.

### HTML-версия

Те же изменения: в `public/bitrix/html/index.html` пересобрать `.hero` под две колонки + новый блок категорий; в шапке (`header` partial внутри каждого `*.html`) убрать выпадающий каталог, добавить поиск. CSS — обновить `public/bitrix/css/hero.css` и `header.css`.
